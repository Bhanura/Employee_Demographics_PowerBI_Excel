# Employee Demographics Power BI Dashboard

![Dashboard Summary](images/summary_dashboard.png)

## üöÄ About This Project

This project is an interactive Power BI dashboard designed to analyze and visualize the demographics of a 100-employee company. The goal is to provide HR and management with actionable insights into workforce composition, tenure, salary distribution, and more.

## üìä Dashboard Features

The report is broken into two main pages:

### 1. Summary Dashboard
A high-level overview of the entire workforce, focusing on:
* **Key KPIs:** Total Employee Count, Average Age, Average Tenure, and Average Salary.
* **Core Demographics:** Distributions by Gender, Age Group, and Marital Status.
* **Organizational View:** Headcount by Department and Position.
* **Geographic Distribution:** A map visual showing employee counts by country.

### 2. Detailed Analysis
A granular, cross-filterable view for deeper analysis, including:
* **Departmental Deep Dive:** Analyze gender balance, average age, and education levels within each department.
* **Compensation Analysis:**
    * Average Salary by Department and Position.
    * A scatter plot visualizing the relationship between *Years of Service* and *Salary*.
    * Average Salary by Education Level.
* **Tenure Breakdown:** A histogram showing the distribution of employee tenure (Years of Service).
